HTML Fundamentals

HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and content of a webpage.

I have learned about:

  • Basic Structure and Tags: Understanding the foundational structure of HTML documents, including the <html>, <head>, and <body> tags, as well as common elements like headings, paragraphs, lists, and links.
  • Semantic HTML: Utilizing semantic HTML elements such as <header>, <nav>, <section>, <article>, <footer>, and <aside> to provide meaning and structure to web content, enhancing accessibility and SEO.
  • Forms and Inputs: Creating forms using <form> elements and various input types such as text, email, password, checkbox, radio buttons, and submit buttons to collerct user input.
  • HTML5 Features: Exploring new features introduced in HTML5, including the <video> and <audio> elements for multimedia content, <canvas> for dynamic graphics, <svg> for scalable vector graphics, and <localStorage> for client-side data storage.
CSS Styling

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML.

I have learned about:

  • Selectors and Specificity: Understanding CSS selectors and specificity to target HTML elements and apply style effectively.
  • Box model and layout: Exploring the CSS box model and techniques for controlling the layout and positioning of elements on a webpage.
  • Flexbox and Grid: Utilizing Flexbox and Grid layouts for creating flexible and responsive designs with ease.
  • Responsive Design: Implementing responsive design principles using media queries and viewport units to create websites that adapt to different screen sizes and devices.
VS Code Mastery

Visual Studio Code (VS Code) is a source code editor developed by Microsoft. It includes support for debugging, embedded Git control, syntax highlighting, intelligent code completion, snippets, and code refactoring.

I have learned about:

  • Installation and Setup: Installing Visual Studio Code and configuring settings for an optimal development environment.
  • Productivity Tips and Shortcuts: Learning useful shortcuts and productivity features in VS Code to stramline the development workflow.
  • Extensions for Web Development: Exploring and installing extensions in VS Code to enhance functionality for web development tasks such as HTML, CSS, and JavaScript.
GitHub for Version Control

GitHub is a platform for hosting and sharing code using Git version control. It provides collaboration features such as issue tracking, pull requests, and project management tools.

I have learned about:

  • Setting up a Repository: Creating a new repository on Github and initializing it with README file.
  • Branching and Merging: Understanding branching strategies and merging changes between branches using Git and Github.
  • Collaboration Workflows: Collaborating with others on projects using pull requests, code reviews, and issue tracking features of Github.
Website Deployment

Website deployment involves making a website accessible to the public by hosting it on a server. Various methods such as GitHub Pages, web hosting services, and continuous deployment strategies are used for website deployment.

I have learned about:

  • Deploying to Codespace: Hosting our website using Codespace, a web hosting platform that provides scalable and customizable hosting solutions for web applications.
  • Utilizing Web Hosting Services: Deploying your website on commercial web hosting services such as AWS, DigitalOcean, or Netlify for more control customization.
  • Continuous Deployment Strategies: Implementing continous deployment pipelines using tools like GitHub Actions or Jenkins to automate the development process whenever changes are pushed to the repository.
Linux Essentials

Linux is an open-source operating system kernel that forms the basis of many different distributions (distros). Understanding Linux basics is essential for managing web servers and development environments.

In this section, you'll learn about:

  • Command-line Basics: Mastering essential comman-line operations such as navigating directories, listing files/directories, and managing permissions.
  • File System Navigation: Navigating the File Linux files system hierarchy using commands like cd, ls, pwd, and navigating between directories.
  • Package Management Managing software packages using package managers like APT(Advance Package Tool) or YUM(Yellow Updater Modified) to install, update, and remove software packages efficiently.
